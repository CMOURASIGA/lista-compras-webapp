Política de Privacidade

Este aplicativo coleta dados limitados dos usuários com o objetivo exclusivo de autenticação e personalização da experiência.

Dados Coletados

  • Nome e endereço de e-mail vinculados à conta Google
  • Identificador único de usuário (Google ID)
  • Dados relacionados às ações feitas no app (ex: listas salvas)

Uso das Informações

As informações coletadas são utilizadas para:

  • Permitir login com a conta Google
  • Associar as informações ao usuário de forma segura
  • Personalizar a interface e funcionalidades conforme o perfil do usuário

Nenhuma informação é vendida, compartilhada ou repassada a terceiros.

Armazenamento

Os dados são armazenados em servidores seguros e acessíveis apenas pelo desenvolvedor responsável.

Direitos do Usuário

O usuário pode solicitar a exclusão de seus dados a qualquer momento, enviando um e-mail para cmourasiga@gmail.com.
